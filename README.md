# Odd Even Checker package published at https://pypi.org/project/odd-even-checker-viswas/

A simple Python package to check if a number is odd or even.

can be imported using pip install odd-even-checker-viswas
# Usage

```python
from odd_even_checker import is_even, is_odd

print(is_even(10))  # True
print(is_odd(7))    # True
